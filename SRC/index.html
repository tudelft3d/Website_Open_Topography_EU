Welcome to the European Topography Elevation Searcher

This interactive map allows you to explore publicly available Digital elevation models and point cloud datasets across European countries. Click on a country to view detailed metadata, acquisition information, and links to downloadable sources.

The website is based on point cloud dialects in europe, inventarisation of point cloud datasets in europe for a uniform framework by D.H. van der Heide.
